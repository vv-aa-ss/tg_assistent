# TG Assistant Bot

## Быстрый старт

1. Установите зависимости:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2. Создайте файл `.env` рядом с `requirements.txt` и укажите переменные:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ADMIN_IDS=123456789,987654321
DATABASE_PATH=./data/bot.db
```

3. Запуск бота:

```bash
python -m app.main
```

## Возможности

- Админ-панель: Карты, Пользователи, Привязать пользователя, Привязать сообщение
- Перешлите администратором сообщение пользователя боту — бот подберёт реквизиты:
  - по привязке пользователя
  - по шаблону текста (подстрока или regex)

## Хранение администраторов

- ID администраторов указываются в `.env` в переменной `ADMIN_IDS`.
- Сам `.env` не коммитьте в репозиторий.
